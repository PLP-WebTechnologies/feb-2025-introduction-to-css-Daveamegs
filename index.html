Introduction to CSS

Learn how to style your web pages with CSS.

Learn More

What is CSS?

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML or XML.

CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.

Why Use CSS?

CSS allows you to separate content from design, making it easier to maintain and update your web pages.

With CSS, you can control the layout, colors, fonts, and overall look of your website.

Getting Started with CSS

To start using CSS, you can either include it directly in your HTML file using the <style> tag or link to an external CSS file using the <link> tag.

Here is an example of how to link to an external CSS file:

            <link rel="stylesheet" href="style.css">

Once you have linked your CSS file, you can start adding styles to your HTML elements.

For example, to change the color of all <h1> elements to blue, you can use the following CSS rule: 

            
                h1 {
                    color: blue;
                }

This will apply the blue color to all <h1> elements on your page.

CSS also allows you to use classes and IDs to target specific elements. For example, you can add a class to an element like this: 

            <h1 class="main-title">Hello World</h1>

And then style it in your CSS file like this:

            
                .main-title {
                    color: red;
                    font-size: 2em;
                    text-align: center;
                }

This will change the color of the <h1> element with the class main-title to red, set its font size to 2em, and center it on the page.

Styling an Image

To style an image, you can use the img selector in your CSS file. For example:

            
                img {
                    width: 100%;
                    height: auto;
                    border-radius: 10px;
                }

This will make all images on your page responsive, meaning they will scale with the size of the screen, and give them rounded corners.

CSS also supports media queries, which allow you to apply different styles based on the screen size or device type. This is essential for creating responsive designs that look good on all devices.

For example, you can use a media query to change the background color of your page on smaller screens: 

            
                @media (max-width: 600px) {
                    body {
                        background-color: lightblue;
                    }
                }

This will change the background color to light blue when the screen width is 600 pixels or less.

In conclusion, CSS is a powerful tool for styling your web pages and creating beautiful designs. With just a few lines of code, you can transform the look and feel of your website.

Start experimenting with CSS today and see how it can enhance your web development skills!
